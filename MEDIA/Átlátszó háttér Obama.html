You said:
Youtube videó htmlbe
ChatGPT said:

Ha szeretnél egy YouTube-videót beágyazni egy HTML oldalba, itt van egy egyszerű példa:

html
<!DOCTYPE html>
<html lang="hu">
<head>
  <meta charset="UTF-8">
  <title>YouTube Videó Beágyazása</title>
  <style>
    .video-container {
      position: relative;
      padding-bottom: 56.25%; /* 16:9 arány */
      height: 0;
      overflow: hidden;
      max-width: 100%;
    }
    .video-container iframe {
      position: absolute;
      top: 0;
      left: 0;
      width: 100%;
      height: 100%;
      border: 0;
    }
  </style>
</head>
<body>

  <h1>Beágyazott YouTube Videó</h1>

  <div class="video-container">
    <iframe 
      src="https://www.youtube.com/embed/VIDEÓ_AZONOSÍTÓ"
      title="YouTube videó lejátszó"
      allowfullscreen>
    </iframe>
  </div>

</body>
</html>

🔧 Csere: A VIDEÓ_AZONOSÍTÓ helyére írd be a YouTube-videó azonosítóját, amit a videó URL-jéből tudsz kivenni.
Például:
YouTube link: https://www.youtube.com/watch?v=dQw4w9WgXcQ
Az azonosító: dQw4w9WgXcQ

Szeretnéd, hogy beállítsam neked egy konkrét videóval? Csak dobj be egy linket.
